# **App Name**: MedConnect+

## Core Features:

- Emergency SOS: One-tap SOS sends SMS/email to emergency contacts with geo-location. Also displays nearest hospitals/doctors with beds.
- AI Symptom Analysis: AI-powered chat analyzes symptoms and suggests potential diagnoses and doctor recommendations using a Gemini tool. Includes medical image analysis for rashes/wounds.
- Doctor/Hospital Search: Search and filter doctors/hospitals by specialty, location, and availability, displayed on a map.
- Appointment Booking: Book appointments with doctors via calendar and time slots.
- Real-time Hospital Bed Tracking: Tracks real-time availability of hospital beds/ICU/ventilator via a map and displayed list.
- User Dashboard: Personalized health dashboard displays appointments, health trends, alerts, and medical history. Syncs data with a Firestore database.
- Firebase Authentication: User authentication and secure storage of user data in Firestore. Uses Firebase Authentication.

## Style Guidelines:

- Primary color: Medical green (#00A676), providing a sense of trustworthiness and health.
- Background color: Desaturated green (#E0F2F7), creates a calm, clean, and professional look.
- Accent color: Red (#FF6B6B), prominently used for emergency buttons and critical alerts to instantly draw attention.
- Body and headline font: 'PT Sans' for clear and accessible reading across devices.
- Use medical and location-based icons for quick navigation.
- Card-based design for clear information hierarchy.
- Subtle animations for a smooth user experience when loading content and during transitions.
